<상품 관리 서비스>
상품 목록은 상품의 ID, 상품명, 가격, 수량 을 확인할 수 있다
상품 등록 버튼을 눌러 상품을 목록에 추가할 수 있다
상품 목록은 상품 ID, 상품명을 클릭시 상품 상세 페이지로 넘어갈 수 있다

상품 상세 화면은 해당 상품의 정보를 확인할 수 있으며, 상품 정보를 수정할 수 있다

상품 등록 폼은 상품명, 가격, 수량 의 정보를 받아 상품을 등록할 수 있다

상품 수정 폼은 상품 ID 를 제외한 상품명, 가격, 수량 정보를 수정할 수 있다
상품 수정이 완료되면 상품 상세 페이지로 전환되어야 한다

<추가된 요구사항>
판매 여부는 판매 오픈 여부를 체크 박스로 선택할 수 있다.

등록 지역은 서울, 부산, 제주가 있으며 체크 박스로 다중 선택할 수 있다.

상품 종류는 도서, 식품, 기타가 있으며 라디오 버튼으로 하나만 선택할 수 있다.

배송 방식은 빠른 배송, 일반 배송, 느린 배송 셀렉트 박스로 하나만 선택할 수 있다.

<검증 요구사항>
타입 검증 - 가격, 수량에 문자가 들어가면 검증 오류 처리

필드 검증 - 상품명: 필수, 공백X / 가격: 1000원 이상, 1백만원 이하 / 수량: 최대 9999

특정 필드의 범위를 넘어서는 검증 - 가격 * 수량의 합은 10,000원 이상

<Bean Validation 수정 요구사항>
등록시에는 `quantity` 수량을 최대 9999까지 등록할 수 있지만 수정시에는 수량을 무제한으로 변경할 수 있다
등록시에는 `id` 에 값이 없어도 되지만, 수정시에는 id 값이 필수이다
